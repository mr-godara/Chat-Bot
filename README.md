## Chat Connoisseur Bot

### Topic of this chatbot

The topic of this chatbot is an AI-powered conversational assistant with mission-based guidance. It is designed to provide interactive, goal-focused chat support through a clean and modern web interface.

### Why I chose this topic

I chose this topic because chatbots are highly relevant in real-world applications and combine both frontend development and AI interaction. This project helped me practice building user-friendly interfaces, structured conversations, and practical product features in one system.
